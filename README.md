### Enable script API
https://script.google.com/home/usersettings


### first off, install then login (globally)

Install
```js
npm install -g @google/clasp
```

Ver check
```
clasp -v
```

Login
```
clash login
```

### clone to local
terminal is on `vscode-to-gas`

```
clasp clone your_project_id --rootDir=src
```

### push to cloud
terminal is on `vscode-to-gas`
```
clasp push
```

### open gas on cloud
terminal is on `vscode-to-gas`
```
clasp open
```
